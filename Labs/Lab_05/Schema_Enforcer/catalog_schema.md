# Catalog (Normalized) Table Schema

| Column Name | Required Data Type | Brief Description |
| :--- | :--- | :--- |
| `instructor_name` | `VARCHAR(100)` | Instructor full name. |
| `instructor_role` | `VARCHAR(50)` | Instructor role (e.g., Primary, TA). |
| `course_id` | `VARCHAR(20)` | Course identifier (e.g., DS2002). |
| `section` | `VARCHAR(10)` | Course section identifier (e.g., 001). |
| `title` | `VARCHAR(200)` | Full course title. |
| `level` | `INT` | Course level as integer (e.g., 200, 300). |
